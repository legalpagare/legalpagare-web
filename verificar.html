¿Cómo funciona el Sello de Autenticidad?
🔐

Código único

Cada serie de pagarés genera un código SHA-256 único basado en los datos del documento.

📧

Fecha cierta

El código se envía por email a deudor y acreedor, estableciendo una fecha cierta e irrefutable.

✓

Verificación pública

Cualquier persona puede verificar la autenticidad de un documento escaneando el QR o ingresando el código aquí.
